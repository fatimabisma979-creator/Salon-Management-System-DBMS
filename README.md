#  Salon Management System (DBMS Project)

## Project Overview
This is a desktop-based database management system built using **Microsoft Access**. It was developed as a group project for our 3rd-semester DBMS course to streamline salon operations like customer bookings, service tracking, and billing.

## 🛠 Features
- **Client Management:** Store and retrieve customer contact details and history.
- **Appointment Scheduling:** Manage daily bookings and assign stylists to specific services.
- **Service Inventory:** A categorized list of salon services (Hair, Skin, Nails) with pricing.
- **Billing & Reports:** Automated calculation of total costs and generation of simple receipts.

## 📊 Database Design
The database follows relational design principles to ensure data integrity.
- **Tables include:** `Customers`, `employee`, `Appointments`, `Services`, 'payment', 'product', 'branch', 'salonusers' and `Payments`.
- **Key Constraints:** Enforced Referential Integrity and Cascade Updates.

## 🚀 How to Use
1. Download the `.accdb` file from this repository.
2. Open it using **Microsoft Access 2016** or later.
3. Enable "Content" if prompted to allow the forms and queries to run.
